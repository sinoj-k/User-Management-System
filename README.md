# User Management System - Spring Boot

## Description
A RESTful User Management System built using Spring Boot that supports CRUD operations.

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST API

## Features
- Create user
- Get user by ID
- Update user
- Delete user
- Global exception handling

## API Endpoints
POST /UMS/addUser  
GET /UMS/getUser
GET /UMS/getAllUsers/{id}
PUT /UMS/updateUser/{id}  
DELETE /UMS/deleteUser/{id}

## How to Run
1. Clone repository
2. Configure MySQL in application.properties
3. Run `UserManagementApplication.java`
